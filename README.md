# School Management System

Backend system developed with a focus on organization, best practices, and clean architecture, simulating a real-world school management environment.

---

## Technologies Used

* Node.js
* Express
* JavaScript
* MVC Architecture

---

## About the Project

The **School Management System** was built to practice fundamental backend development concepts, such as:

* Separation of concerns (MVC)
* Layered architecture
* Route and controller creation
* Simulation of real-world business rules
* Structure ready for database integration

---

## Project Structure

```bash
src/
 ├── controllers/
 ├── models/
 ├── routes/
 └── app.js
```

* **controllers** → application logic
* **models** → data structure
* **routes** → route definitions
* **app.js** → main server configuration

---

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/GuilhermeVidal25/Sistema-Escolar.git
```

### 2. Navigate to the folder

```bash
cd Sistema-Escolar
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the server

```bash
npm start
```
